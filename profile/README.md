## About

Puppet Digital is an independent studio built around a single principle: focused work produces better software. Operating lean and moving deliberately, every product ships with its name on it.

Each project is owned end-to-end — design, engineering, testing, and deployment. No hand-offs, no overhead. The result is software that is coherent, performant, and built to last.
